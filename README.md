# Courier Tracking Website

## Overview
A web-based courier tracking system developed as an academic project. The application allows users to track parcels, view shipment status, and explore courier services through a user-friendly interface.

## Features
- Parcel Tracking
- Shipment Status Updates
- Customer Support
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- Bootstrap
- JavaScript
- PHP
- MySQL
